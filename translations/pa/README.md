> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git

## 🌱 ਸ਼ੁਰੂਆਤ ਕਰਨਾ

ਇਸ ਕੋਰਸ ਵਿੱਚ AI ਏਜੰਟ ਬਣਾਉਣ ਦੀਆਂ ਬੁਨਿਆਦੀ ਗੱਲਾਂ ਦੀ ਪੜਾਈ ਹਨ। ਹਰ ਸਬਕ ਆਪਣੇ ਵਿਸ਼ੇ ਨੂੰ ਕਵਰ ਕਰਦਾ ਹੈ ਇਸ ਲਈ ਤੁਸੀਂ ਜਿੱਥੇ ਚਾਹੋ ਸ਼ੁਰੂ ਕਰੋ!

ਇਸ ਕੋਰਸ ਲਈ ਬਹੁ-ਭਾਸ਼ੀ ਸਹਾਇਤਾ ਹੈ। ਸਾਡੇ [ਉਪਲਬਧ ਭਾਸ਼ਾਵਾਂ ਇੱਥੇ ਤੱਕ](#-multi-language-support) ਜਾਓ।

ਜੇ ਤੁਹਾਡੀ ਪਹਿਲੀ ਵਾਰੀ ਹੈ ਜੈਨਰੇਟਿਵ AI ਮਾਡਲਾਂ ਨਾਲ ਬਣਾਉਣ ਦੀ, ਤਾਂ ਸਾਡੇ [ਜੈਨਰੇਟਿਵ AI ਫਾਰ ਬਿਗਿਨਰਜ਼](https://aka.ms/genai-beginners) ਕੋਰਸ ਵੇਖੋ, ਜਿਸ ਵਿੱਚ GenAI ਨਾਲ ਬਣਾਉਣ ਲਈ 21 ਸਬਕ ਹਨ।

ਇਸ ਰੀਪੋ ਨੂੰ [ਸਟਾਰ (🌟) ਕਰਨਾ ਨਾ ਭੁੱਲੋ](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ਅਤੇ ਕੋਡ ਚਲਾਉਣ ਲਈ ਇਸ ਨੂੰ [ਫੋਰਕ ਕਰੋ](https://github.com/microsoft/ai-agents-for-beginners/fork)।

### ਹੋਰ ਸਿੱਖਣ ਵਾਲਿਆਂ ਨਾਲ ਮਿਲੋ, ਆਪਣੇ ਸਵਾਲਾਂ ਦੇ ਜਵਾਬ ਪ੍ਰਾਪਤ ਕਰੋ

ਜੇ ਤੁਹਾਨੂੰ ਕਦੇ ਫਸਾਉਟ ਹੋ ਜਾਂ AI ਏਜੰਟ ਬਣਾਉਣ ਬਾਰੇ ਕੋਈ ਸਵਾਲ ਹੋਵੇ, ਤਾਂ ਸਾਡੇ ਨਿਰਧਾਰਤ Discord ਚੈਨਲ ਵਿੱਚ ਸ਼ਾਮਿਲ ਹੋਵੋ [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) ਵਿੱਚ।

### ਤੁਹਾਨੂੰ ਕੀ ਚਾਹੀਦਾ ਹੈ

ਇਸ ਕੋਰਸ ਦਾ ਹਰ ਸਬਕ ਕੋਡ ਦੇ ਉਦਾਹਰਣਾਂ ਦੇ ਨਾਲ ਆਉਂਦਾ ਹੈ, ਜੋ code_samples ਫੋਲਡਰ ਵਿੱਚ ਮਿਲ ਸਕਦੇ ਹਨ। ਤੁਸੀਂ [ਇਸ ਰੀਪੋ ਨੂੰ ਫੋਰਕ ਕਰ ਸਕਦੇ ਹੋ](https://github.com/microsoft/ai-agents-for-beginners/fork) ਆਪਣੀ ਕਾਪੀ ਬਣਾਉਣ ਲਈ।

ਇਹਨਾਂ ਅਭਿਆਸਾਂ ਵਿੱਚ ਕੋਡ ਉਦਾਹਰਣ Microsoft Agent Framework ਨਾਲ Azure AI Foundry Agent Service V2 ਵਰਤਦੇ ਹਨ:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure ਖਾਤਾ ਲੋੜੀਂਦਾ

ਇਸ ਕੋਰਸ ਵਿੱਚ Microsoft ਤੋਂ ਹੇਠਾਂ ਦਿੱਤੇ AI ਏਜੰਟ ਫਰੇਮਵਰਕ ਅਤੇ ਸੇਵਾਵਾਂ ਵਰਤੀ ਗਈਆਂ ਹਨ:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


ਇਸ ਕੋਰਸ ਲਈ ਕੋਡ ਚਲਾਉਣ ਬਾਰੇ ਹੋਰ ਜਾਣਕਾਰੀ ਲਈ, [ਕੋਰਸ ਸੈਟਅਪ](./00-course-setup/README.md) 'ਤੇ ਜਾਓ।
